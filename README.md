# Planning/Scrum Poker with LARAVEL
Yazılım projelerinde yapılacak işlerin veya geliştirilecek yeni özelliklerin ne kadar zor, karmaşık veya zaman alıcı olduğunu tahmin etmek (estimate) için kullanılan bir sistemdir. Ekip üyelerinin, birbirlerinin fikirlerinden etkilenmeden, bir işin zorluğunu özgürce puanlamasını sağlar.
